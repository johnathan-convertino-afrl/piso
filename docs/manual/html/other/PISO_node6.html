In a Project

This core is made to interface for parallel to serial data. Data is loaded into the core, and then enable is pulsed to push data out in a serial fashion. There is a count output that tells how many bits have been output. Data is always presented at the output so when count is at 31, the serial data output has bit 31 presented at the output. The series of steps to use it are as follows.

  1. Set pdata to input data, and set load to 1 (data will be loaded on positive clock edge to internal registers).
  2. Set load to 0.
  3. Pulse enable to 1 that is synced to the main clock. Only hold high for one period of the master clock.
  4. Repeat enable pulse untill dcount is equal to 0.
